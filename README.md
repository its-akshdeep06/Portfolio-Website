# Akshdeep Singh - Portfolio Website

A personal portfolio website built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**. Live at [akshdeep06.vercel.app]([https://akshportfolio-sage.vercel.app/](https://akshdeep06.vercel.app/)).

---

## Overview

This portfolio showcases Akshdeep Singh's academic background, technical skills, projects, and contact information. It is designed as a single-page application with smooth section navigation and a clean, modern UI.

---

## Features

- **Responsive single-page layout** with anchor-based navigation (Home, About, Skills, Projects, Contact)
- **About section** covering academic journey, current learning stack, future vision, and personal interests
- **Skills section** highlighting languages and technologies in progress
- **Projects showcase** — including [MindfulCare](https://mindfulcare01.vercel.app/) (a mental health web app) and [Kanban Board](https://kanban-aksh.vercel.app/) (a drag-and-drop task management tool)
- **Contact section** with email, university email, phone, GitHub, and Instagram links
- **Dark mode support** via Tailwind's `darkMode: 'class'` configuration
- **Deployed on Vercel** for fast, global delivery

## Getting Started

### Prerequisites

- Node.js ≥ 18
- npm ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/akshdeep06/Portfolio_Website.git
cd Portfolio_Website

# Install dependencies
npm install
```

## Deployment

The project is deployed on **Vercel**. Any push to the `main` branch triggers an automatic redeployment.

To deploy your own fork:

1. Import the repository at [vercel.com/new](https://vercel.com/new)
2. Vercel auto-detects Vite — no extra configuration needed
3. Click **Deploy**
